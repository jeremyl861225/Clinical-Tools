Fournier's Gangrene Scoring Tool

FGSI / UFGSI

富尼氏壞疽嚴重度指數 · Fournier's Gangrene Severity Index（Laor 1995）與 Uludağ 修訂版（Yilmazlar 2010）
兩個分數的關係、用途與限制 Definitions
◆ FGSI（Laor 1995）＝ APACHE II 急性生理分數中九項參數的加總（體溫、心率、呼吸速率、血鉀、血鈉、肌酸酐、血比容、白血球、碳酸氫根），各項 0–4 分，滿分 36 分。原始研究 30 例，切點 >9：死亡機率 75%；≤9：存活機率 78%。

◆ UFGSI（Yilmazlar 2010，Uludağ 大學）＝ FGSI（生理分數 a）＋侵犯範圍分數 b＋年齡分數 c，滿分 43 分。原始研究 80 例（死亡率 21%），切點 >9：死亡機率 94%；≤9：存活機率 81%。ROC 分析顯示 UFGSI 優於 FGSI（p = 0.002）。

◆ 侵犯範圍分數（b）：侷限於泌尿生殖／肛門直腸區 = 1；侷限於骨盆區 = 2；超出骨盆區（延伸至腹壁、大腿、後腹腔）= 6。
◆ 年齡分數（c）：≥60 歲 = 1；<60 歲 = 0。

◆ 外部驗證：Üreyen 2017（土耳其兩中心 29 例）以 ROC 求得最佳切點 FGSI >6（敏感度 100%、特異度 78.3%）與 UFGSI >8（敏感度 100%、特異度 73.9%），並指出侵犯範圍是最重要的預後參數之一。Yilmazlar 亦建議 UFGSI >8 者應入加護病房。

◆ 限制：Roghmann 2012（BJU Int，44 例前瞻＋回溯）比較 FGSI、UFGSI、年齡校正 Charlson 共病指數（ACCI）與 surgical Apgar，四者 AUC 差異均不顯著；作者認為 UFGSI 雖納入較多變項，預測力未優於 FGSI。
本分數為預後／分流工具，不是手術與否的判準——富尼氏壞疽一經診斷即需緊急徹底清創，分數低不得作為延後手術的理由。

◆ 資料時點：原始研究以入院當時的生理數值計算。
◆ 本工具的邊界處理：原始表格逐格印為「>41 / 39–40.9 / …」，格與格之間存在小數空隙（如 38.4 與 38.5 之間）。本工具依 APACHE II 的連續邊界實作（≥41、≥39、≥38.5…），使任何數值都能落入唯一級距，與原表在整數與一位小數上完全一致。

FGSI 滿分 36 分，UFGSI 滿分 43 分。上述死亡率來自各原始世代（30–80 例的單中心研究），族群小、年代與治療條件不同，應視為風險分層而非個人化預測。本分數不決定是否手術：富尼氏壞疽一經診斷即需緊急徹底清創（WSES 2018 recommendation 1C），分數低不得作為延後手術或縮小清創範圍的理由。未填入之生理欄位以 0 分（正常）計。
