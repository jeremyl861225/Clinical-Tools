急性肺栓塞 PE

Acute Pulmonary Embolism — 診斷路徑、危險分層、抗凝與再灌流（2019 ESC）
kg 供 enoxaparin／alteplase 換算

診斷路徑取決於一件事：病人穩不穩。血流動力學不穩者不做機率評分，直接找右心與栓子； 穩定者才走「臨床機率 → D-dimer → CTPA」的階梯，避免不必要的顯影劑與輻射。

兩條分流
不穩定
休克／低血壓SBP < 90 或需升壓
視為高危 PE。立即床邊心臟超音波找右心室負荷（RV 擴大、D-sign、McConnell）；病人若太不穩無法送 CT，超音波見右心負荷即可經驗性再灌流。能穩住就緊急 CTPA 確診。
穩定
臨床機率評分Wells / Geneva
用 Wells 或修訂 Geneva 分「PE 不太可能 vs 可能」。可搭配 PERC（低機率者八項全陰可免驗 D-dimer）。
穩定
D-dimer（年齡校正）
非高機率者驗 D-dimer；> 50 歲用年齡校正臨界值（年齡 ×10 µg/L）提高特異度、減少不必要 CT（ADJUST-PE）。YEARS 演算法可進一步減少造影。陰性即排除、不做 CT。
穩定
CTPA（或 V/Q）確診
高機率、或 D-dimer 陽性 → CT 肺動脈攝影。顯影劑禁忌／腎功能差／孕婦可改 V/Q 掃描。孕婦依 YEARS-pregnancy 調整，必要時先照下肢超音波。
Wells／Geneva／PESI 等分數本站另有計算器可用（見「計分工具」）。D-dimer 在住院、癌症、老年、術後偽陽性高，陽性只代表「需要影像」而非確診。

確診後立即分層——它決定要不要再灌流、在哪裡照護。四個軸：血流動力學、sPESI、右心功能（超音波／CT）、心肌指標（troponin）。

2019 ESC 四級分層
高危 High
血流動力學不穩（休克／持續低血壓／需升壓）。無論其他指標 → 再灌流。早期死亡最高。
中高危 Interm.-High
穩定，但 右心功能異常「且」troponin 上升。密切監測，惡化即救援再灌流。
中低危 Interm.-Low
穩定，右心異常「或」troponin 上升（僅其一，或皆輕微）。住院抗凝。
低危 Low
穩定、sPESI = 0、無右心負荷、troponin 正常。可考慮早期出院／門診治療。
分層要件
  • sPESI（簡化 PESI）任一項為 1 即 ≥ 1：年齡 > 80、癌症、慢性心肺病、脈搏 ≥ 110、SBP < 100、SpO₂ < 90%。0 分屬低危。
  • 右心功能：超音波 RV/LV > 1、TAPSE 下降、McConnell 徵象；或 CT 上 RV/LV > 1。
  • 心肌指標：troponin 上升、BNP/NT-proBNP 升高提示右心受累。
早期出院／門診治療的條件（低危）
Hestia 準則或 sPESI = 0，且無右心負荷、社會支持足、能取得口服抗凝、無其他住院理由。用 DOAC 銜接最方便。
依分層決定強度
高危
全身溶栓（優先）± 導管／外科取栓
血流動力學不穩 → 立即全身纖溶（alteplase），同時 UFH 抗凝、循環支持（謹慎補液、norepinephrine 升壓、必要時 VA-ECMO）。
  • 溶栓禁忌或失敗 → 導管取栓／導管定向溶栓，或外科取栓。
  • 心臟停止疑似 PE 所致 → 復甦中可考慮溶栓。
中高危
抗凝 ＋ 密切監測
先 抗凝並嚴密監測，不常規全身溶栓（PEITHO：常規溶栓減少血流動力學惡化，但增加大出血與顱內出血）。惡化（休克化）即救援再灌流。導管定向治療（HI-PEITHO／PEERLESS 系列）是新興選項，仍在累積證據。
中低／低危
抗凝為主
直接口服抗凝（DOAC）為多數病人首選；低危可早期出院／門診。
抗凝的起始與療程
起始選擇
  • DOAC 優先（除非禁忌）：apixaban 或 rivaroxaban 可直接口服起始（有前段高劑量期）；dabigatran／edoxaban 需先注射抗凝 5 天再轉換。
  • 腸外起始：LMWH（enoxaparin）或 fondaparinux；高危、需溶栓、極重腎衰竭、血流不穩用可快速調控的 UFH。
  • 特殊族群：抗磷脂症候群、機械瓣 → 用 warfarin（DOAC 不適）；癌症相關 → DOAC 或 LMWH（腸胃道／泌尿道腫瘤留意出血）；懷孕 → LMWH（DOAC、warfarin 禁）。
療程
誘發性（可逆因子，如術後）至少 3 個月後停；無誘因或持續危險因子（癌症、復發）考慮延長／終生，並定期評估出血與復發風險。

體重相關者（enoxaparin、alteplase）依上方體重換算；DOAC 為固定劑量。
