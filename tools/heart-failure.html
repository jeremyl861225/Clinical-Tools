壹、心臟衰竭之定義、病因和預後 Definition, epidemiology and prognosis

心臟衰竭是一種臨床症候群，其特徵是由於心臟結構和／或功能異常，引起休息或負荷時心輸出量減少和／或心內壓力增高，從而導致典型症狀（呼吸困難、足踝水腫、疲倦感），也可伴有徵象（頸靜脈壓升高、肺部囉音、周邊水腫）。

因為心臟衰竭為臨床導向之診斷，在出現明顯臨床症狀之前，患者可表現為無症狀的心臟結構或功能異常（如收縮或舒張左心室功能不全），此為心臟衰竭之前兆。及早發現此類前兆很重要，因為此類病患預後較差須及早介入，進而有效降低死亡率。

右心室衰竭 Right ventricular dysfunction

心臟衰竭也可以是單純右心室功能異常導致，可能致病機轉包括壓力超載（pressure overload）或容積超載（volume overload）。雖然造成慢性右心室衰竭較常見的原因為左心衰竭後的肺高壓（LV dysfunction-induced pulmonary hypertension），但仍有許多原因可造成右心室衰竭，例如心肌梗塞、心律失常性右心室心肌病變（ARVC）或心臟瓣膜疾病。

診斷上需要定量性評估整體右心室功能，例如 fractional area change (FAC)、TAPSE，或 Doppler tissue imaging-derived systolic S' velocity of the tricuspid annulus (TA S')。

貳、心臟衰竭之診斷 Diagnosis of Heart Failure

心臟衰竭症狀通常是非特異性的，所以無助於區別心臟衰竭及其他問題。由於體液滯留引起的症狀和表徵，用利尿劑治療可以快速緩解。諸如頸靜脈壓升高和心尖搏動移位等表徵比較特異性，但難於檢查出且重複性差。肥胖個體、老年和慢性肺病患者的症狀和表徵可能特別難以鑑別和解釋。

應當收集詳細的病史。在沒有相關病史（如心臟損傷的潛在原因）的個體中，心臟衰竭是不常見的。每次訪視時都要評估症狀和表徵，特別是注意鬱血的證據；儘管治療但症狀持續存在，通常表示需要另外的治療，而症狀不斷惡化是一種嚴重的進展（置患者於緊急住院和死亡的風險），需要及時就醫。

診斷慢性心臟衰竭之五個步驟 2021 ESC · 5 steps

一、心電圖

若心電圖為正常結果則心臟衰竭較不可能。心房顫動、Q 波、左心室肥大、QRS 較寬會增加心臟衰竭的可能性，也可能指引後續治療方向。

二、Natriuretic peptide

若 BNP < 35 pg/mL 或 NT-proBNP < 125 pg/mL 則較不像是心臟衰竭。

三、基本抽血檢測

肝腎功能、電解質、全血細胞計數、甲狀腺功能。用於鑑別診斷、提供預後資訊以及指引治療方向。

四、心臟超音波

評估心臟功能的重要檢查。除 LVEF 外，還能了解心腔大小、左心室肥大程度、局部心肌收縮障礙、右心室衰竭、肺高壓、瓣膜功能以及舒張功能。

五、胸部 X 光

可鑑別其他造成喘的病因，並提供心衰竭的證據如肺部鬱血或心臟肥大。

鑑別潛在病因之進階檢查 Advanced work-up
一、經食道心臟超音波 TEE
對心臟衰竭的常規診斷評估是不需要的；然而在心臟瓣膜疾病、疑似主動脈異常、疑似心內膜炎或先天性心臟病，以及需要心臟復健的心房顫動患者（為了排除心房內血栓）可能具有價值。當單用 TTE 時二尖瓣或主動脈瓣病變嚴重度與症狀不相符，應當做 TEE。
二、負荷心臟超音波 Stress echo
運動或藥物負荷心臟超音波可用於評估所誘發的心肌缺血和／或心肌存活率，某些情況下可評估瓣膜疾病（動力性二尖瓣逆流、低血流–低壓力梯度主動脈狹窄）。對於活動性呼吸困難、LVEF 保留、休息時舒張功能參數不能做結論的患者，可檢測出與運動相關的舒張功能不全。
三、心臟核磁共振 CMR
公認測量左右心室容量、質量和射血分數的黃金標準。心臟超音波未能診斷（特別是右心成像）時的最佳替代模式，複雜先天性心臟病之首選。LGE 與 T1 mapping 是評估心肌纖維化之首選；可區別缺血性與非缺血性病因，並確認心肌炎、類澱粉沉積症、結構病、查加斯病、法布瑞氏症、緻密化不全心肌病變和血鐵沉積症的組織特徵；亦可評估心肌缺血與心肌活力。
四、單光子電腦斷層掃描與心室核子攝影
可用於評估心肌缺血和心肌活力；門控可提供心室容量與功能資訊，但會使患者暴露於輻射。DPD 或 PYP 閃爍掃描可用於檢出甲狀腺運送蛋白心肌類澱粉沉積症（ATTR）。
五、正子攝影掃描 PET
單用或與 CT 結合，可用於評估心肌缺血和心肌活力。
六、冠狀動脈血管攝影
對藥物難治型心絞痛的心衰患者（其他方面適合血管再灌流）建議做；有症狀的心室心律異常史或複雜心臟停止史者也建議；冠心症前測可能性中到高度、非侵入性負荷檢查存在缺血者，應考慮以明確缺血性病因與嚴重度。
七、次世代定序 NGS
對於符合心肌病變診斷標準的患者建議進行基因檢測，以促進診斷、預後評估和治療分層，同時對親屬進行階層式基因評估。
參、低收縮分率心臟衰竭 Heart Failure with reduced ejection fraction

診斷 HFrEF 需要有心臟衰竭的症狀或徵象，以及較低的左心室收縮分率（LVEF ≤ 40%）。心臟超音波是最常使用的工具；EACVI 建議的測量方式包括線性測量（2D M-mode）、容積測量（apical 4/2 chamber）、超音波顯影劑輔助測量或 global longitudinal strain。若無法使用超音波測量，少數狀況下可運用心臟磁振造影或核子掃描。

治療目標
  1. 降低死亡率 reduction in mortality
  2. 預防因心衰竭惡化而住院 prevention of recurrent hospitalizations
  3. 改善臨床症狀、功能耐受度及生活品質 clinical status, functional capacity and QOL
藥物治療為 HFrEF 的治療基石，且治療順序上應先藥物治療、後儀器治療（device therapy）。

核心藥物治療建議 Pharmacologic therapy A–E
A. 血管張力素轉化酶抑制劑 ACEi

ACEi 為最早能降低 HFrEF 病患死亡率的藥物，並且也能改善臨床症狀。除非有禁忌症或無法忍受，建議使用於所有病患並調高劑量至最大可忍受之治療劑量。

禁忌症

  1. 曾有血管性水腫病史 History of angioedema
  2. 已知雙側腎動脈狹窄
  3. 懷孕或有懷孕風險
  4. 已知有過敏反應或其他不良反應（drug-specific）

應特別留意或應尋求專科協助之族群

  1. 顯著的高血鉀（serum K+ > 5.0 mmol/L）
  2. 顯著的腎功能不全（Cr > 2.5 mg/dL 或 eGFR < 30 mL/min/1.73 m²）
  3. 有症狀低血壓或無症狀嚴重低血壓（SBP < 90 mmHg）
  4. 潛在藥物交互作用：鉀離子補充劑、保鉀利尿劑、MRA、腎素抑制劑、NSAID、TMP-SMX、標榜低鹽之調和物（含較多鉀離子替代）

使用方式

  1. 給藥前確認腎功能及電解質，用藥後 1–2 週再次追蹤，達最終劑量後 1–2 週再追蹤。
  2. 低劑量起始；門診病患每兩週以上可雙倍增加劑量，住院或可密切追蹤者可更頻繁加倍。
B. 乙型阻斷劑 β-blockers

已證實在使用 ACEi 之下，能減少有症狀 HFrEF 病人住院及死亡之風險，並改善臨床症狀。ACEi 與 β-blocker 可以在確診時就一起開立，目前沒有研究證據指出兩者先後選擇有所差別。因急性心臟衰竭住院者，一旦病情穩定，建議低劑量開始並逐步提高至可忍受之最高劑量。

禁忌症

  1. 無心臟節律器之二度或三度房室傳導阻滯
  2. 危急性肢體缺血
  3. 氣喘：為相對禁忌症。使用心臟選擇性 β-blocker 時氣喘並非絕對禁忌，但應留意是否需停藥。COPD 不是禁忌症。
  4. 已知有過敏反應或其他不良反應

應特別留意之族群

  1. 嚴重心臟衰竭（NYHA class IV）
  2. 現在或近期（四週內）有心衰竭惡化、房室傳導阻滯、或心率 < 50 bpm
  3. 若持續出現鬱血徵象、低血壓（SBP < 90 mmHg）、頸靜脈壓上升或顯著周邊水腫 → 先治療鬱血並達到 euvolemia 再給藥
  4. 潛在藥物交互作用（增加心搏過緩／房室阻滯風險）：verapamil/diltiazem（不建議繼續使用）、毛地黃、amiodarone、ivabradine

使用方式

  1. 低劑量起始；門診病患每兩週以上可雙倍增加，有些病患需要更久間隔。
  2. 以達最大或最大可接受劑量為主，並注意心律、血壓及臨床徵狀（特別是鬱血徵象與體重）。
C. 醛固酮受體拮抗劑 MRAs

對於 LVEF ≤ 35% 的病人，使用 ACEi 及 β-blocker 仍不能控制症狀時，MRA 能降低住院及死亡風險 Class I / A。Eplerenone 相比 spironolactone 較不會造成男性女乳症。當腎功能異常或血清鉀 > 5 mmol/L 時，需密切追蹤腎功能及鉀離子。

D. 血管張力素受體–腦啡肽酶抑制劑 ARNI

2014 年 PARADIGM-HF 後，ARNI 比 enalapril 更能降低心衰惡化住院率、心因性死亡及整體死亡率；此外可改善生活品質、降低糖尿病需要胰島素之比例、降低腎功能惡化及高血鉀風險，並可能降低利尿劑需求。雖較常出現有症狀低血壓，這群病患仍能得到好處。

  • Ambulatory HFrEF 使用最佳藥物治療後仍有症狀 → 可考慮將 ACEi/ARB 更換為 ARNI。
  • 2019 PIONEER-HF、TRANSITION：住院中病患（含未用過 ACEi/ARB 的 de novo）起始 ARNI 可降低 42% 後續 CV death 或心衰竭住院 → de novo HFrEF 可考慮使用 ARNI Class IIb / Level B。
  • 若曾使用 ACEi，建議至少 36 小時洗滌期（wash-out）再使用 ARNI，以降低血管性水腫風險。
E. 第二型鈉–葡萄糖共同轉運蛋白抑制劑 SGLT2i
  • DAPA-HF（2019）：ambulatory HFrEF 於最佳藥物治療外加 dapagliflozin，下降 26% 初級終點（worsening HF 或 CV death）；亦降低 all-cause mortality、緩解症狀、改善活動功能與生活品質。好處在治療初期即可觀察到，且無論有無糖尿病或 HbA1c 高低皆有益。
  • EMPEROR-Reduced（2020）：NYHA II–IV 之 HFrEF 加上 empagliflozin，下降 25% 初級終點（CV death 或 HF hospitalization）；納入 eGFR > 20 mL/min/1.73 m² 病患，並觀察到降低腎功能惡化速率。該研究未見 CV mortality 顯著下降，但兩試驗之 meta-analysis 顯示 CV mortality 並無異質性。

因此 HFrEF 病患無論有無糖尿病，已使用最佳藥物治療（ACEi/ARNI、β-blocker、MRA）後，建議加上 dapagliflozin 或 empagliflozin；額外好處是可降低利尿劑劑量。

副作用
可能增加反覆性生殖器黴菌感染風險。剛起始時可能輕微降低腎功能，但此降低可預期且可逆，不需因此中止治療。
可考慮使用於特定病患之藥物 Selected patients
A. 利尿劑 Diuretics　Class I / Level C

建議用於改善體液過剩、增進運動耐受性及降低心衰竭住院率。Loop 類相較 Thiazide 類提供較強且短效的利尿效果；合併使用兩者有加乘效果並可改善利尿劑阻抗性（diuretic resistance），但也增加不良作用風險，須小心使用。此外 ARNI、MRA 及 SGLT2i 也有部分利尿效果。目標：以最低劑量維持 euvolemia，可依症狀及體重調整。

B. 血管張力素受體阻斷劑 ARBs　Class I / Level B

目前建議使用於已接受 β-blocker 及 MRA，但因嚴重副作用而無法忍受 ACEi 或 ARNI 的病人，以降低心衰竭住院率及心因性死亡率。Candesartan 被證實能減少心因性死亡，valsartan 能改善因心衰竭住院的機率。使用時應注意低收縮壓、腎功能不全、或血清鉀 > 5 mmol/L；除非禁忌或無法忍受，應從低劑量逐步調整至最大可忍受劑量。

C. If-channel 抑制劑 Ivabradine

藉由抑制竇房結 If-channel 降低心律，只能使用在正常竇性心律的病人。

  • 有症狀 HFrEF（LVEF ≤ 35%）、竇性心律、靜止心率 ≥ 70 bpm，且過去 12 個月曾因心衰竭住院、已接受最佳藥物治療（ACEi/ARB、β-blocker、MRA）→ 可考慮 Class IIa / Level B
  • 同上條件但無法忍受或禁忌 β-blocker，已使用 ACEi/ARNI 及 MRA → 可考慮 Class IIa / Level C
D. Hydralazine + Isosorbide dinitrate
  • African American 且有症狀（NYHA III–IV），已接受最佳藥物治療後仍有症狀 → 降低住院及死亡率 Class I / Level A 第三版提升
  • 有症狀 HFrEF 無法忍受 ACEi / ARB / ARNI → 可考慮以降低死亡風險 Class IIb / Level C
E. 毛地黃 Digoxin　Class IIb / Level B

1997 年 DIG 研究發現毛地黃對死亡率是中性的。治療區間較窄，血清濃度應維持在 0.5–< 0.9 ng/mL 第三版下修。

使用於老人、脆弱病患、低血鉀、腎功能不全、併用 amiodarone 新增及營養不良病患中應格外小心。

有症狀之 HFrEF、竇性心律並已接受最佳藥物治療（ACEi 或 ARNI、β-blocker、MRA）後，可考慮使用以減少整體住院率（包括心因性住院）。

肆、低收縮分率心臟衰竭之節律治療 Cardiac rhythm management

體內去顫器 Implantable cardioverter-defibrillator, ICD

心臟衰竭的病人有高比例會猝死，原因包含心血管事件、心室心律不整、心搏過緩及心跳停止，其中部分事件和電解質不平衡相關。可改善或延緩心血管疾病進展的治療雖能降低平均每年猝死機率，但對心律不整效果仍有限。ICD 可有效預防心搏過緩以及校正潛在致命之心律不整；相比之下有些抗心律不整藥物雖可能降低心搏過速之心律不整及猝死，但無法降低、甚至可能增加整體死亡率。

A. 猝死之次級預防 Secondary prevention

與 amiodarone 相比，ICD 可降低「經歷心跳停止的存活者」以及「持續性有症狀之心室心律不整」病人之死亡率。放置需考慮病人意願、生活品質、心室收縮功能（LVEF > 35% 的病人好處仍未知）以及預期壽命大於一年以上。

ESC 建議：出現造成血行動力學不穩之心室心律不整，且預期餘命超過一年、功能狀態良好、無可逆病因，或為心肌梗塞後大於 48 小時之病患 → 放置 ICD 可降低猝死風險及整體死亡率 Class I。

B. 猝死之初級預防 Primary prevention

心衰竭的猝死比例自 1990 至 2015 年間下降了 44%，主要歸功於治療進步及指引建議治療（β-blocker、MRA、ARNI、CRT-P）。

抗心律不整藥物
2005 SCD-HeFT：NYHA II–III 之 HFrEF 使用 amiodarone 無法改善整體死亡率，反而是單導極 ICD 比安慰劑或 amiodarone 更能降低整體死亡率。長期使用 amiodarone 應格外小心。Dronedarone 或第一類抗心律不整藥物（disopyramide、encainide、flecainide）會增加死亡率，不該用於心衰竭病患預防心律不整。

缺血性心肌病變比非缺血性有較高猝死機會，ICD 絕對效益也較高（Class I vs Class IIa）。臨床研究也發現心肌梗塞後 40 天內放置 ICD 並無好處，因此不建議在 40 天內放置作為初級預防。

ESC 建議：有症狀之心衰竭（NYHA II–III），先接受至少三個月最佳藥物治療後，LVEF 仍未改善超過 35%，且預期餘命超過一年、功能狀態良好 → 可評估置入 ICD。缺血性 Class I；非缺血性 Class IIa。

C. 病患選擇 Patient selection

低收縮分率心衰竭合併心電圖 QRS > 130 ms 則應放置 CRT-D 而非一般 ICD。

ICD 對於嚴重共病多、預期餘命小於一年的病人幫助有限；也不建議使用在過於嚴重的心衰竭病患（NYHA class IV）、對藥物治療無效且不適合 CRT、VAD 或心臟移植的病人 Class III。

放置前應與病患討論：放置目的、去顫器運作之相關併發症、關閉機器之時機（因末期疾病）或移除之時機（因感染、左心室功能恢復）。

D. 皮下式去顫器 S-ICD

與傳統經靜脈 ICD 治療效率相似，併發症發生率也相似。即使 S-ICD 在放置初始有較高的不當電擊，適當病患選擇後 S-ICD 不劣於經靜脈 ICD。在困難靜脈管路或有感染風險的病患是較好的選擇。但 S-ICD 無法治療緩慢性不整脈，也無法提供 anti-tachycardia pacing (ATP) 或 CRT，因此病患選擇上要格外小心。

診斷 HFmrEF 需要有心臟衰竭之症狀或徵象，以及輕微下降的收縮分率（41–49%）。若有上升之 NPs（BNP ≥ 35 pg/mL 或 NT-proBNP ≥ 125 pg/mL），或其他結構異常之證據（左心房擴大、左心室肥大或超音波測量左心填充）可提高診斷可能性；但如果可以準確測量收縮分率，這些間接證據則非必要。

臨床特徵
相比 HFpEF，HFmrEF 的臨床表現與 HFrEF 較為相似：男性較多、年紀較輕、較高機率有冠心症（約 50–60%），且較少有心房顫動及非心臟性共病。但在死亡率上 HFmrEF 與 HFpEF 較為相似，無症狀的 HFmrEF 死亡率低於 HFrEF。HFmrEF 族群包含由 < 40% 進步至此、或由 ≥ 50% 降低至此的病患。
治療 Treatment

與其他族群相同，利尿劑可用於治療體液過剩之症狀。過去針對 LVEF > 40% 缺乏具強烈實證的改善預後藥物（ACEi/ARB/ARNI/β-blocker/MRA 建議等級僅 Class IIb）；然而隨著 EMPEROR-Preserved 與 DELIVER 發表，證明了 SGLT2 抑制劑在這類族群的效益。

陸、正常收縮分率心臟衰竭 HFpEF · Diagnosis & treatment

「正常（preserved）」一詞最早主要出現在 2003 年 CHARM 研究中，相對於收縮分率低於 40% 的病患。以收縮分率分類心臟衰竭仍有許多爭議：中等收縮分率是否該用 41–49% 劃分？男女之間是否該用相同標準？目前歐洲心血管影像學會定義收縮失常為男性 LVEF < 52%、女性 < 54%。

此外，若定義正常收縮分率是高於 50%，那極高收縮分率（例如 > 65–70%）也應尋求是否有特定病因，例如 cardiac amyloidosis 或肥厚性心肌病變；而「超正常 supra-normal」的收縮分率也可能是因為左心室舒張末容積縮減（公式中的分母減少）導致。

臨床特徵
HFpEF 病患年紀較大、較常為女性、較常有心房顫動及慢性腎病，以及其他非心血管性共病。

應懷疑心臟澱粉沉積症的線索：有高血壓病史卻出現 low normal BP、對 β-blockers 或 ACEi 治療不耐受、雙側腕隧道症候群病史、心電圖低電位（low voltage），以及典型心超特徵（縱隔／後壁／右心室增厚、心房擴大、少量心包膜積液、瓣膜增厚）。

另有許多疾病表現相似應及早排除：肺部疾病、貧血、肥胖或健康狀況下降（deconditioning）。
診斷標準 Diagnostic criteria

診斷 HFpEF 較為困難。近期有兩個以分數為主的診斷流程（H₂FPEF 及 HFA-PEFF），但兩者皆為中等似然度（intermediate likelihood），需要額外確認工具強化診斷。

HFpEF 之治療 Treatment

先前針對 HFpEF 的大型臨床研究包含 PEP-CHF(perindopril)、CHARM-Preserved(candesartan)、I-PRESERVE(irbesartan)、TOPCAT(spironolactone)、DIG-Preserved(digoxin) 及 PARAGON-HF(sacubitril/valsartan)。其中 candesartan 及 spironolactone 可觀察到降低心衰竭住院率之趨勢，但仍未達初級終點。2005 SENIORS 中 nebivolol 雖顯著降低全因死亡及心衰竭住院率，但納入 LVEF > 50% 的病患僅佔全體 15%，轉譯至 HFpEF 仍需謹慎。PARAGON-HF 次分析發現 LVEF < 57% 者使用 ARNI 可降低心因性死亡及心衰竭住院率；TOPCAT 中亦觀察到 LVEF < 55% 者有類似結果。

  • 積極治療潛在病因與心血管／非心血管共病 Class I：高血壓、冠心症、心臟澱粉沉積症、心房顫動、心臟瓣膜疾病等 — 治療這些潛在疾病也可改善 HFpEF 之臨床症狀
  • Finerenone：2025 日本指南建議有症狀之 HFpEF 可考慮使用 Class IIa / Level B（尚未納入歐美指引）
柒、心衰竭和懷孕 HF and Pregnancy

孕期的心肌病變可能來自孕前的心肌病變或因懷孕引起的心肌病變。周產期心肌病變定義為懷孕晚期或產後早期的心肌收縮功能障礙，通常 LVEF < 45%，常伴有左心室擴張。

危險因子

年紀 > 30 歲、非洲血統、多胎、多產、子癇前症／子癲症、貧血、糖尿病、肥胖、延長孕期。

預後相關因子

初診斷時左心室收縮功能、左心室血栓是否形成、整體右心室被影響的程度、子癲前症、地理種族。

流行病學與風險
  • ROPAC（Registry of Pregnancy and Cardiac disease）分析 1321 位結構性心臟病女性：妊娠前或周產期心肌病之女性死亡率約 2.4%。
  • 發生率約千分之一，是懷孕期間死亡的主要原因；推測與自體免疫、擴張性心肌病變病史有關。早期症狀常與正常懷孕生理現象類似而容易被忽視。
  • 懷孕是高血液凝固狀態，合併左心室收縮功能下降易併發心臟內血栓，發生率約 16%。抗凝血劑為標準治療；EF < 30% 時考慮在分娩後 6–8 週使用抗凝血劑（此時高凝固狀態最明顯）。若使用 bromocriptine 治療，建議合併抗凝血劑。

常見臨床表現包括疲勞、陣發性夜間呼吸困難、水腫、頸靜脈充血，且因孕期凝血因子活化容易形成血栓栓塞。NYHA 分級常以第 III 或 IV 級症狀最常見，有時病程變化快速，會在幾天之內進展到心因性休克、心律不整、多重器官衰竭。

處置重點在於確定診斷、改善心衰竭症狀、確保胎兒安全、減少心衰竭惡化，因此孕前諮詢及共同討論與決策相當重要。治療需考量孕期、是否哺乳來調整藥物；若心肺狀態不穩應考慮儘快引產。非藥物措施著重於降低液體和鹽分攝取。

A. 病患衛教
良好的心臟衰竭衛教，包括自我評估（self-monitoring）、自我調整（self-management），可更即時改善生活品質、降低再住院率以及死亡率。內容包括：了解心臟衰竭常見之臨床症狀、目前的治療方式及策略、目前使用藥物之適應症／併發症／副作用、日常活動／睡眠狀況／水份攝取之醫療建議，以及定期接種疫苗、避免酒精或戒菸對於心臟衰竭之好處。
B. 心肺運動
透過運動可改善運動耐受度，進而改善生活品質及降低心臟衰竭住院率。許多研究發現高強度間歇運動可能改善病患之最高氧氣攜帶率（VO₂）。指引建議在合併有較嚴重共病或衰弱症之 HFrEF 病患，可考慮接受有指導下的運動導向心肺訓練 Class IIa。
C. 病患追蹤

應定期評估臨床症狀、治療療效，以及早發現無症狀之心臟衰竭惡化。指引建議追蹤頻率不應超過六個月，且每次追蹤應評估臨床症狀、心跳、血壓、白血球分析、電解質及腎功能。若病患剛出院或正在上調藥物劑量，返診頻率應提高。

BNP、NT-proBNP 雖然是很好的臨床預後指標，但使用生物指標去指引藥物治療仍有爭議，目前指引並不建議常規使用 BNP 或 NT-proBNP 來指引治療。

科技輔助器具（智慧型手機、穿戴式智慧手錶）可提供心跳、心律、血壓等數據。隨著 COVID-19 改變醫療模式，居家醫療評估（home telemonitoring）工具可幫助臨床醫師及早發現病情變化並及早介入。

處理優先順序與病人配置 Triage & disposition

急性心臟衰竭病況變化大又快，診斷與評估需在第一次接觸病人或其資訊時及早開始。危及生命之臨床狀況或需緊急治療的情況為最優先處理；一開始若具有需要急救、心因性休克或呼吸衰竭的狀況，需要緊急處理及立即安排 CCU 或內科加護病房住院。診斷的第一步通常是排除其他可能造成症狀的原因（肺部感染、嚴重貧血、急性腎衰竭）。

A. 住院準則

  1. 持續且嚴重呼吸喘或血行動力不穩病人，應檢傷至可立即提供復甦支持的地點。
  2. 高風險病人（持續且嚴重呼吸喘或血行動力不穩、反覆心律不整、合併急性冠心症），初始治療應在加護病房。
  3. 入住加護病房準則（以下任一）：
    • 須插管或已插管
    • 有灌流不足症狀或徵象
    • 給予氧氣下仍血氧濃度不足（SpO₂ < 90%）
    • 使用輔助肌呼吸或呼吸速率 > 25/min
    • 心跳 < 40 bpm 或 > 130 bpm、收縮壓 < 90 mmHg
  4. 其他病人通常須入住普通病房；只有少數（通常鬱血症狀輕微）能在給予利尿劑和調整口服藥物後直接從急診出院。臨床狀況穩定後可從加護病房轉入普通病房，後續治療應包括多專業團隊。

B. 辨識須立即處理的加重因子

1. 急性冠心症
應依相關治療指引處理。合併急性冠心症與急性心臟衰竭之病人是風險很高的一群，建議盡快進行侵入性治療以達成血管疏通。
2. 高血壓危症
因快速血壓升高而加重心臟衰竭者常以肺水腫表現，首要治療目標為盡快降壓，建議以針劑血管擴張劑加利尿劑積極降壓（在前幾個小時降幅 25%）。
3. 頻脈、嚴重緩脈或傳導異常
合併嚴重心律異常且狀況不穩時，應以藥物、電擊或暫時性心律調節器緊急矯正。若心律不整造成血行動力失償，建議以電擊整流回復竇性心律。處理急性心臟衰竭合併持續心室心律不整非常有挑戰性，因兩者為惡性循環交互作用；某些個案可考慮血管攝影（如需要則血管疏通）或電生理檢查以進行電燒。
4. 急性機械性因素
可能表現為急性冠心症之併發症（游離壁破裂、心室中膈缺損、急性二尖瓣逆流）、胸部外傷、心臟侵入性治療、因心內膜炎造成急性瓣膜缺損、主動脈剝離、血栓或其他罕見阻塞原因（如腫瘤）。診斷需仰賴心超，治療通常要藉助循環支持系統和外科或介入性治療。
5. 急性肺栓塞
應以相關治療指引處理；若確立肺栓塞為休克或低血壓原因，建議立即進行手術或導管溶栓以達成再灌流。

C — CLASSIC臨床證據顯示存在組織灌流不足，且初步需要藥物或機械性循環支持。通常會伴隨低血壓症狀。
D — DETERIORATING臨床證據顯示休克正在惡化，或即便已升級治療方案，病情仍未能改善。
E — EXTREMIS難治性休克狀態，或已發生／即將發生急性循環衰竭。

2. 掌握休克辨識

臨床症狀

意識障礙／意識狀態改變、皮膚蒼白且濕冷、冒冷汗、呼吸急促／呼吸困難、心跳過速。

血流動力學徵象

  • 心臟指數 CI < 2.2 L/min/m² 或心輸出量 CO < 4.0 L/min
  • 心臟功率輸出 CPO < 1.0（CPO < 0.6 定義為重度心因性休克）
  • 終末器官灌流不足之內在證據，例如血乳酸持續升高

3. NTUH 休克團隊啟動流程

啟動方式
滿足以下兩類條件時，由心臟加護病房專任醫師啟動或經心臟內科會診後啟動。

一、三項中至少符合兩項

  • 血壓異常：SBP < 90 mmHg 或 MAP < 65 mmHg 持續達 30 分鐘
  • 需使用血管加壓劑、強心劑或機械性循環支持（MCS）以維持 SBP ≥ 90 mmHg 或 MAP ≥ 65 mmHg
  • 組織灌流不足：末梢器官受損之臨床證據（少尿／無尿、四肢冰冷、意識狀態變差）；或 2 小時內追蹤之 lactate > 2 mmol/L。侵入性血流動力學數據不佳：CI < 2.2 L/min/m² 或 CPO < 0.6 W

二、心肺功能停止後狀態

心肺功能停止患者於 30 分鐘內恢復自主循環（ROSC），且同時具備：

  • 神經功能恢復狀態：GCS ≥ 8 分，或氣切／插管狀態下配合指令者（5T）

初始診斷及評估 Initial work-up

第一步是完整的病史（症狀、過去心血管疾病史、可能的心因性因素或其他非心因性加重因子）；身體檢查可評估是否有鬱血或灌流不足；所有病人都應接受一系列完整檢查（心電圖、胸部 X 光、實驗室檢查、心臟超音波）來確立診斷。

A. 胸部 X 光
特異表現包括肺靜脈鬱血、肋膜積水、間質性或肺泡性水腫、心臟肥大。仰躺的 X 光對診斷用處較少。可排除如肺炎等其他非心因性疾病。
B. 心電圖
在急性心臟衰竭鮮少正常，可協助找出潛在心因性因素或其他加重因子（如快速的心房顫動、急性心肌梗塞）。
C. 心臟超音波
兩類病人有必要做立即的心超：血行動力不穩（特別是心因性休克）；懷疑有危及生命的急性心臟結構或功能異常（機械性併發症、急性瓣膜逆流、主動脈剝離）。所有初次發生急性心臟衰竭或心臟功能不明的病人都應考慮早期完成心超，目前以住院後 24 小時內完成為目標。易快速惡化之心臟疾病（急性心肌炎、大範圍嚴重心肌梗塞）宜於短時間內重複執行。
D. 床邊超音波（胸腹部）
可評估腹水、肋膜積液、間質性肺水腫、下腔靜脈直徑大小，輔助體液狀態和呼吸之評估；亦為置放管路（如中央靜脈導管）、床邊穿刺引流的指引工具。
E. 實驗室檢查

初始檢查應包含 CBC、血糖、基本電解質（鈉、鉀、鎂）、肝功能、腎功能（BUN、Creatinine）、凝血功能（aPTT、PT）。

  1. 利鈉尿肽（NPs）：本院檢查項目為 NT-proBNP，具高敏感性，若為正常值則急性心臟衰竭可能性很小（正常值：NT-proBNP < 300 pg/mL）。然而升高並不能確立診斷（見表 8-2）。有些失償末期心臟衰竭、右心衰竭或突發性肺水腫病人可能有預料外的低利鈉尿肽濃度。
  2. 心肌旋轉蛋白（troponin）：本院為 troponin-T，所有病人都建議抽取以偵測急性冠心症。若有符合指引的動態變化則需積極排除 ACS；但大多數心衰竭病人即使無 ACS 心臟酵素仍會升高，表示持續性的心肌受損，判讀時需特別小心。
  3. 動脈血氣體分析：不須常規檢測，適用於無法藉由血氧計監測氧合，或需精確量測氧氣與二氧化碳分壓時。靜脈血分析可推測 pH 值和二氧化碳分壓。
  4. Procalcitonin 或 CRP：用於懷疑合併感染時；procalcitonin 特別適用於鑑別診斷肺炎及導引治療。
  5. D-dimer：喘的病人用於懷疑肺栓塞。敏感性高但特異性低；數值正常可用來排除肺栓塞，陰性預測率高達 95% 以上。

A. 氧氣治療與呼吸器支持

不建議在無低血氧病人常規給予氧氣，因為會造成血管收縮而降低心輸出量。在 COPD，高氧合可能加重通氣與灌流差異，進而抑制呼吸造成高碳酸血症。氧氣治療期間須監測酸鹼平衡與血氧。

非侵襲性正壓呼吸器包括 CPAP 與 BiPPV；BiPPV 可給予吸氣壓力支持、增加每分通氣量，特別適合高碳酸血症病人（COPD 最為典型）。

鬱血會影響肺功能和增加肺內分流造成低血氧。除非有禁忌症，氧分壓依需要（根據 SpO₂）應上調至 100%，但應注意避免高氧症。非侵襲性正壓呼吸器可減少呼吸作功、插管和死亡率；到院後仍有呼吸衰竭徵象病人應繼續使用；若有酸血症或高碳酸血症者以 PS-PEEP 為宜。

鎮靜藥物
Propofol 具有心臟抑制性可能導致低血壓；midazolam 較少心臟相關副作用。因此急性心臟衰竭與心因性休克者，較適合使用 midazolam。
B. 藥物治療 — 1. 利尿劑 第三版改寫

利尿劑是急性心臟衰竭合併體液過量與鬱血症狀病人之治療基石，可增加腎臟鹽分與水分排除，並有些微血管擴張效果。

重要順序
若病人合併灌流不足症狀，應避免在達成足夠灌流前給予利尿劑。

若血壓許可，處理鬱血的第一步包括針劑利尿劑與血管擴張劑：

  • 如果門診已有使用環利尿劑 → 建議使用 1–2 倍的劑量，靜脈給予
  • 若為新診斷或過去未使用利尿劑者 → 可從單次注射 furosemide 20–40 mg 開始
  • 給予後需追蹤尿量、urinary spot sodium 評估反應，適時調整劑量，並留意腎功能及電解質
  • 可併用不同種類利尿劑（acetazolamide、thiazide 類）以達到更佳利尿效果
B-1b. 血管擴張劑
心臟衰竭第二常用之症狀緩解藥物，但無強烈證據證明其好處。可減低靜脈張力（優化前負荷）和動脈張力（減少後負荷），因此可能增加心搏量。對合併高血壓病人尤其有效；若低血壓或有相關症狀者則須避免。須小心控制劑量以避免過度降壓（與預後不佳有關）；若合併主動脈瓣或二尖瓣狹窄者應小心使用。
B-2. 強心劑

使用應限於嚴重心輸出降低而導致標的器官灌流不足者，通常見於低血壓性心臟衰竭；但若低血壓之原因為低血溶或其他可矯正之因素，則不建議在矯正這些因素前使用。強心劑（特別是有腎上腺素作用者）會造成心搏過快，也可能造成心肌缺氧與心律不整，需監測心電圖。過去研究顯示強心劑有可能增加死亡率。須小心使用、從低劑量開始，並在密切監測下調整劑量。

3. 升壓劑（血管收縮）

給予顯著低血壓之病人具周邊動脈收縮效果藥物（norepinephrine 和高劑量 dopamine），可提高血壓以增進到標的器官之血流，但代價是增加左心室後負荷。有一研究次族群分析顯示 norepinephrine 比 dopamine 的副作用少且死亡率較低。Epinephrine 應用於在藥物使用下仍持續低血壓或急救時。

4. 栓塞預防

若無禁忌症建議以 heparin 或其他抗凝血劑預防栓塞。

5. Vasopressin 受體拮抗劑

Tolvaptan 抑制 arginine vasopressin 在腎小管 V2 受體的作用，促進水排出（aquaresis）；可用於體液過多又嚴重低鈉的病人。

6. Opioid 鴉片類

可緩解焦慮與喘，不建議常規使用，只可在嚴重呼吸喘病人審慎考慮。劑量相關副作用包括噁心、低血壓、心搏過緩、呼吸抑制。因其可能有增加死亡率之風險，使用上仍有爭議。

7. 抗焦慮與鎮靜藥物

焦躁或譫妄病人可能需要；小心使用 benzodiazepine（diazepam 或 lorazepam）可能是較安全的做法。

C. 機械輔助裝置

1. 主動脈氣球幫浦 IABP

傳統適應症為急性機械性問題手術前、嚴重心肌炎、心肌梗塞或嚴重心肌缺氧於導管或手術前中後之血循支持。目前指引不建議「常規」使用於急性心衰竭病人，無明確證據顯示 IABP 在其他心因性休克之好處。但在缺血性心臟病需增加冠狀動脈灌流支持、高風險 PCI 病人及心因性休克需增加心臟支持時，仍扮演重要角色。主要併發症：出血、置於肢末端灌流不足、氣球破裂、位置不正確影響主動脈分枝血流、主動脈破裂。

2. 心室輔助器 VAD

心室輔助器或其他機械循環支持可作為特定病人之階段性治療或長期治療。

心絞痛跟冠心病 Angina & CAD

A. 藥物治療

Beta-blocker 跟 ivabradine 是控制心絞痛症狀的有效藥物，也是 HFrEF 心臟衰竭治療的重要藥物。Amlodipine、nicorandil 跟 nitrates 被證明使用在 HFrEF 病人是安全的。Diltiazem 跟 verapamil 則被認為使用在 HFrEF 病人是不安全的。

B. 重新建立心肌血流 Revascularization

透過心導管或外科方式重新建立血流可改善缺血性心衰竭患者的預後。根據 ESC 治療指引，當病人有多條血管疾病及可接受的手術風險下，繞道手術為首選。選擇心導管或外科方式應由內外科心臟團隊根據臨床狀況、血管構造、預期重建血流的完整度、是否合併瓣膜疾病及其他共病做整體評估。

惡病質與肌少症 Cachexia & sarcopenia

惡病質是一個影響全身組織的消耗性過程。約 5–15% 心臟衰竭病人會發生，且隨疾病嚴重度惡化，跟症狀嚴重程度、功能下降、住院率增加、存活率降低都有關聯。

定義：在 6–12 個月內有大於體重 6% 非水腫相關的體重減輕。

可能機轉包括發炎免疫反應的活化、神經內分泌失調、營養不良吸收不佳、攝取的熱量跟蛋白質失衡、代謝趨力（anabolic drive）降低、長時間無法行動等。約 30–50% 的 HFrEF 病人會出現骨骼肌消耗跟活動能力下降。

治療包括促進食慾的藥物、運動復健、加強代謝的藥物（testosterone），再合併營養補充跟抑制分解代謝的治療。不過目前關於好處跟安全度皆尚未有定論。

癌症 Cancer & cardiotoxicity

許多治療癌症的藥物會導致或惡化左心收縮功能跟心臟衰竭，最常見的是 anthracyclines（doxorubicin）、trastuzumab 跟 tyrosine kinase inhibitors。最近一篇 Cochrane review 發現 dexrazoxane 在有使用 anthracyclines 的病人有心臟保護的效果。

若病人接受心臟毒性化療，給藥前後 LVEF 評估、global strain speckling 的評估是被建議的。

罹患乳癌的女性接受 trastuzumab 治療，有個根據年齡、化療藥物、基礎心血管狀況跟其他共病的風險指數表可幫助評估後續出現心臟衰竭的風險。當病人出現中重度的心收縮功能異常，應停止化療並接受心臟衰竭藥物治療；若事後心臟功能有恢復，日後化療的好處跟風險應審慎評估。

縱膈腔的電療也可能產生長期心臟併發症，心臟酵素（NPs、troponins）的追蹤可用來評估跟監測高心臟毒性風險的病人。

中樞神經系統共病 CNS comorbidities

中風跟心臟衰竭因有共同的危險因子故常常共存，兩個都有較差的預後，而中風會讓心臟衰竭的病人更難自理。在高中風風險的病人，抗血小板跟抗凝血的藥物使用需取得平衡。

自律神經失調在嚴重的 HFrEF 病人中尤其常見。再合併低血壓，病人可能會因暈眩而容易受傷，而且許多心臟衰竭的藥物在理想劑量上會有所限制。減低利尿劑可能可以降低姿勢性低血壓的嚴重度。

憂鬱症是常見但常被低估的共病；病患的遵醫囑性較差、社交較為疏離，常有較差的預後。常規使用一些被認可的問卷調查是不錯的方法：The Beck Depression Inventory (BDI) 跟 Cardiac Depression Scale 是目前被認可用來評估心臟衰竭病人憂鬱程度的工具；其他廣泛使用的還有 Geriatric Depression Scale、Hamilton Depression Scale、Hospital Anxiety and Depression Scale。

心理治療、藥物治療跟運動復健對有憂鬱症的 HFrEF 病人都是有效的。認知行為治療跟完整的教育規畫可降低憂鬱嚴重程度、焦慮程度跟疲憊症狀，亦可改善社交功能、心智健康狀況以及心臟衰竭相關的生活品質指標。

SSRIs 被認為是安全的；相對的，因為會導致低血壓、惡化心臟衰竭跟心律不整，應避免使用 tricyclic antidepressants。
糖尿病 Diabetes mellitus

糖尿病是心臟衰竭常見共病，且有較差的預後，但並不影響治療成效 — 舉例來說，無論病人有無糖尿病，beta-blockers 都可以改善預後。嚴格的血糖控制是否可以改善存活率仍未定論。

  • Metformin：跟過去的觀念不同，使用在 HFrEF 病人是安全且有效的。因為會造成乳酸中毒，在肝腎功能不全的病人則依舊是禁忌症。
  • Insulin：對於治療第一型糖尿病、有症狀的高血糖及 beta 胰島疲乏是必要的。然而 insulin 也是一個強效的鈉滯留荷爾蒙，故可能會加重心臟衰竭惡化。
  • Sulfonylurea 衍生物跟心臟衰竭惡化也有關聯，須小心使用。
  • Thiazolidinediones 會導致鈉跟水分滯留，進而增加心臟衰竭惡化風險跟住院率，故應謹慎評估。
  • DPP4i 與 GLP-1 RA：兩者皆可改善血糖，但針對心血管事件風險不降反升且可能使心臟衰竭惡化。更重要的是目前並沒有關於 DPP4i 跟 GLP-1 使用在心臟衰竭病人身上的安全性數據。
  • SGLT2i：一系列臨床試驗顯示可以改善住院率、慢性腎臟病的惡化跟心臟衰竭死亡率，但對心肌梗塞或中風的比率沒有影響。反之，GLP-1 在降低中風的比率較 SGLT2i 明顯。

整體來說，血糖的控制必須依照臨床狀況調整；在增加新的血糖藥後必須密切注意。

勃起障礙 Erectile dysfunction

勃起障礙對心臟衰竭的男性是生活品質重要指標。治療應著重於心血管疾病及重要共病（例如糖尿病）的治療，跟減輕病人焦慮及憂鬱情緒。Thiazide、spironolactone 跟 beta-blockers 可能會加重勃起障礙。

PDE5Is 有正向的血行動力學跟抗心肌重塑的效果，可改善心臟衰竭病人的體適能跟生活品質。然而 PDE5Is 不得跟 nitrates 併用。

痛風跟關節炎 Gout & arthritis

高尿酸血症跟痛風常見於心臟衰竭病人，尤其被利尿劑的使用誘發或加重；高尿酸血症與較差的預後有關。

根據 EULAR 治療指引，當有反覆急性發作、關節病變、痛風石或影像上有痛風的變化，則建議使用降尿酸藥物，目標血中尿酸濃度 < 6 mg/dL。

  • Xanthine oxidase inhibitors（allopurinol、oxypurinol）在心臟衰竭病人的安全性尚未明確，但可以作為痛風的預防。
  • 相較 NSAIDs，秋水仙素（colchicine）為較適合治療急性發作的藥物。
  • 單一關節炎可以使用關節內注射類固醇；全身注射的類固醇則容易造成水分滯留。
  • 關節炎常因自行服用止痛藥（尤其 NSAIDs）而惡化腎臟功能。類風濕性關節炎跟 HFpEF 風險增加有關；DMARDs 使用的安全度尚未被建立。
低血鉀跟高血鉀 Hypo/hyperkalemia

心臟衰竭許多藥物跟高低血鉀有關，而兩者都可能加重心室心律不整。

  • 降血鉀：Loop diuretics 跟 thiazide diuretics
  • 升血鉀：ACEIs、ARBs、MRAs
  • Amiloride 跟 triamterene 有時會作為利尿劑之外的輔助，用來治療頑固型水腫或避免低血鉀
  • 適量食用高鉀食物或服用鉀離子補充劑也可以避免低血鉀
當急性高鉀血症（> 6.0 mmol/L）發生時，保鉀藥物跟 RAAS inhibitors 應暫停；但是後續應盡快小心加回藥物並密切監測血鉀濃度。
高脂血症 Dyslipidemia

LDL 過高在 HFrEF 並不常見，反而在嚴重的病人中 LDL 常常是低的，且有較差的預後。根據兩個大型隨機分派研究，在非缺血性心臟病的心臟衰竭病人使用 rosuvastatin 並不會改善重大後遺症率跟死亡率，但也不會增加風險，且可能可以下降再住院率。

故目前沒有證據建議心臟衰竭病人使用 statin；但若病人因冠心病已經有使用則可以考慮繼續服用。

高血壓 Hypertension

高血壓跟心臟衰竭的發生有關。最近的前瞻性世代研究顯示，心臟衰竭病人中較高的基礎收縮壓、舒張壓跟脈搏壓會有較高比例的不良事件，更進一步支持控制血壓的重要性。

  • 會縮減收縮力（negatively inotropic）的 CCBs 包括 diltiazem 跟 verapamil 應避免使用在 HFrEF；使用在 HFpEF 則被認為是安全的。
  • 若使用 ACEI/ARB、beta-blocker、MRA 跟利尿劑後血壓仍無法控制，hydralazine、amlodipine、felodipine 被認為使用在 HFrEF 是安全的。
  • 高血壓治療指引的目標可以應用在心臟衰竭病人。
  • 在急性心臟衰竭的情況下，靜脈滴注的 nitrates 可用來降血壓。

鐵質缺乏在心臟衰竭中是常見的，並會造成貧血以及骨骼肌失能；若病人有鐵質缺乏則預後較差。在 HFrEF 或 HFmrEF 且合併鐵質缺乏的患者中，建議給予靜脈注射鐵劑，以改善臨床症狀與生活品質 Class I / Level A；亦應考慮使用靜脈鐵劑以降低因心衰竭住院的風險。

貧血定義：男性 Hb < 13 g/dL、女性 < 12 g/dL，在心臟衰竭病人很常見，尤其是住院病人。女性、老人跟慢性腎病變的病人盛行率又更高，且跟心肌重塑、發炎反應以及體液過多有相關。

貧血的心臟衰竭病人會有較嚴重的症狀、較差的功能狀態、較高的住院率以及存活率下降。仍須檢查找出是否可以解釋貧血的原因，包括出血、鐵質缺乏、B12／葉酸缺乏等。

紅血球生成刺激劑並無法改善 HFrEF 病人合併輕度至中度貧血之預後，且會增加栓塞風險，故不建議使用。
腎功能不全（CKD、心腎症候群、攝護腺阻塞） Renal dysfunction

慢性腎病跟心臟衰竭常共存且有較差預後。CKD 定義：eGFR < 60 mL/min/1.73 m²、有蛋白尿。嚴重的腎功能不全（eGFR < 30）因未被納入隨機分派研究，故許多治療缺乏明確證據。

腎臟功能惡化通常指肌酸酐上升超過 0.3 mg/dL 或 25% 增加、或 GFR 下降超過 20%。在急性心臟衰竭住院過程中，肌酸酐的上升並不是每次都有意義（尤其伴隨水腫消去、利尿劑使用、血液濃縮時）。大幅度上升為急性腎損傷則相對少見，常是因為利尿劑合併其他腎毒性藥物接觸（抗生素、顯影劑、ACEI/ARB、消炎止痛藥）。

心臟衰竭病人中腎功能變化相對常見，尤其在開始跟上調 RAAS inhibitor 時，但大部分幅度不大且不須終止治療除非變化太大。當出現後者情況時，應評估是否有腎動脈狹窄、體液過多或過少、其他共存藥物、或高鉀的情形。

Finerenone v3
選擇性非類固醇類鹽皮質激素受體拮抗劑。FIDELIO-DKD 與 FIGARO-DKD 顯示，相較安慰劑 finerenone 能有效降低複合性心血管不良事件（心血管死亡、非致命性腦中風、非致命性心肌梗塞及心衰竭住院）的發生率，且單獨分析心衰竭住院風險時亦顯著降低。因此對於合併 T2DM 與 CKD 的患者，推薦使用 finerenone 以降低因心衰竭住院的風險 Class I / Level A
  • Diuretics 在 GFR 非常低的病人通常藥效較差，劑量應隨之上調。腎臟代謝排出的藥物在腎功能變差時可能累積，需做劑量下調。
  • 有冠狀動脈或周邊血管問題的病人可能因注射顯影劑而有顯影劑引起的急性腎傷害。
  • 攝護腺阻塞在年老男性常見且可能影響腎臟功能。Alpha-adrenoceptor（阻斷劑）會低血壓且易有鈉離子跟水分滯留，使用在心臟衰竭病人可能不安全；5-α-reductase inhibitors 是比較適合的選擇。
肺部疾病 Pulmonary disease

因為症狀容易重疊且肺功能檢查結果不易判斷，在心臟衰竭病人診斷 COPD 跟氣喘是不容易的，常被過度診斷。肺功能檢查應在病人達到穩定跟適當的體液狀態至少三個月後再安排。無論是真正有 COPD 或被誤判，都跟較差的功能狀態及預後有關。

Beta-blockers 在氣喘只是相對禁忌症，而在 COPD 並不是禁忌症。較專一的 beta-1 adrenoceptor antagonist（bisoprolol、metoprolol succinate、nebivolol）較為適合。

Beta-blockers 被認為在氣喘是禁忌症，是根據 1980s–1990s 的研究，當時起始劑量很高且對象為年輕的嚴重氣喘病人。臨床實務上，可考慮以低劑量心臟專一的 beta-blockers 開始並密切注意呼吸道症狀。因此根據 2015 GINA global strategy 並沒有把氣喘列為使用 beta-blockers 的絕對禁忌症；但使用時需非常小心並密切注意病人症狀。

  • 長期使用會刺激心臟的吸入性藥物，其安全性並不確定，在 HFrEF 病人的使用須審慎評估（其在 COPD 跟氣喘的好處主要與症狀緩解有關而非預後）。
  • 口服類固醇會造成鈉離子跟水分滯留，可能惡化心臟衰竭；吸入性類固醇目前不認為有同樣問題。
  • 肺高壓是嚴重 COPD 的長期併發症，可能會引發右心臟衰竭。
  • 非侵入性正壓呼吸可改善因 COPD 合併心臟衰竭引起的高二氧化碳呼吸衰竭。
肥胖 Obesity

肥胖是心臟衰竭的危險因子，且因為一樣會有喘、運動耐受性差、腳腫的情形，還會讓超音波影像品質變差，加上肥胖病人 NP 數值也會上升，故讓診斷變得複雜。相較 HFrEF，肥胖更常見於 HFpEF 病人。

肥胖悖論 Obesity paradox
肥胖雖然是發生心臟衰竭的風險因子，一旦心臟衰竭確診，肥胖卻跟較低的心臟衰竭死亡率有相關。在其他慢性疾病也有類似情形。

若目標是預防未來發生心臟衰竭，肥胖病人應依照 ESC 心血管治療指引處置。然而針對體重較重的心臟衰竭病人不建議減重 — 過去的前瞻性研究並沒有看出好處且安全性也有疑慮，何況在心臟衰竭病程中體重減輕通常預後較差（高後遺症率、高死亡率、症狀惡化、生活品質變差）。

總結：中等程度肥胖的心臟衰竭病人（BMI < 35 kg/m²）不建議減重；重度肥胖的心臟衰竭病人（BMI 35–45 kg/m²）可考慮減重以改善症狀跟體適能。
睡眠障礙及睡眠呼吸障礙 Sleep-disordered breathing

有三分之一的心衰竭病人有睡眠呼吸障礙，其盛行率在急性心臟衰竭惡化的病人中更高。最常見的類型：中樞型睡眠呼吸中止（CSA，類似 Cheyne-Stokes respiration）、阻塞型呼吸中止（OSA）跟兩者的混合。其他造成睡眠障礙的原因還有焦慮、憂鬱、端坐呼吸，以及因利尿劑使用而夜尿。

有證據顯示心臟衰竭病人有 CSA 跟 OSA 會有較差的預後。CSA 跟 HFrEF 有密切的關聯，相互彼此為重要的成因跟常見共病。

睡眠呼吸障礙需做睡眠生理檢查來確診。夜間氧氣支持、CPAP、BiPAP、ASV 在其他治療指引建議可用在治療夜間缺氧（AHI > 30 per hour）的 OSA 病人。然而這些處置對心臟衰竭預後並沒有幫助。

研究顯示 CPAP 可降低心臟衰竭引起的 CSA 病人呼吸中止或缺氧的頻率、改善 LVEF 跟 6 分鐘走路測試，不過對預後或再住院率並沒有幫助。其他新的治療 CSA 方式（例如植入式膈神經刺激）相關研究仍在進行中。

瓣膜性心臟疾病 Valvular heart disease v3 更新

瓣膜性心臟疾病會導致或惡化心臟衰竭。合併心臟衰竭及瓣膜疾病的病人為高風險族群，故關於治療的決策都應該由不同次專科醫師組成的心臟團隊作完整評估。

A. 主動脈狹窄 Aortic stenosis

有嚴重 AS 合併 LVEF 下降的病人屬於 low-flow, low-gradient AS（valve area < 1 cm²、LVEF < 40%、mean pressure gradient < 40 mmHg）。低劑量 dobutamine 壓力超音波測試可以分辨病人為中度或重度狹窄，亦可評估收縮能力。

  • 若 mean gradient > 40 mmHg 且病人有症狀，無論 LVEF 都應該接受主動脈瓣膜置換
  • 在正常收縮分率的低血流、低壓力階差嚴重 AS 亦應接受主動脈瓣膜置換 v3
  • 若經團隊評估不適合開刀且餘命大於一年 → 應考慮 TAVR

B. 主動脈逆流

嚴重主動脈逆流：有症狀病人建議接受主動脈瓣修補或置換；無症狀病人合併左心室功能不全或進行其他心臟手術時應考慮修補或置換。

C. 二尖瓣逆流

Primary（organic）MR：若病人有症狀，或無症狀但有左心室功能不全時，進行 MV 手術。高開刀風險患者應評估進行經導管邊緣對邊修復術（TEER） v3。

Secondary MR：成因為左心室擴大導致二尖瓣關閉不全。有效的指引導向藥物治療可以逆轉心室重塑進而降低功能性二尖瓣逆流；接受 GDMT 後仍有症狀時進行 MV 介入手術，當患者不適合外科手術時應評估進行 TEER。

D. 三尖瓣逆流

次發性／功能性 TR 常見於心臟衰竭的自然病史，主要原因為瓣環擴大。嚴重 TR 會引起跟惡化心臟衰竭症狀；利尿劑可以改善水腫，腹水也常見於這些病人，額外使用 MRA 可能可以改善。

  • 當患者嚴重 TR 需接受左側心瓣膜手術時，建議進行三尖瓣手術
  • 若為原發性嚴重 TR 且有症狀時亦建議三尖瓣手術；不適合手術時進行經導管三尖瓣介入治療（TTVI） v3

A. MCS 於急性心臟衰竭之使用

可支持左心或雙心室衰竭病人至心臟和其他器官功能恢復，通常以幾天至幾周為限。SAVE（Survival After Veno-arterial ECMO）score 可幫助評估病人存活率。MCS（特別是 ECLS 和 ECMO）可用於bridge to decision：先穩定血行動力並回復標的器官功能，再進行完整臨床評估是否心臟移植或轉換至可長期使用之裝置。

B. MCS 於末期心臟衰竭之使用

等待移植的病人在 LVAD 支持下有較高的生存率，因此越來越多 MCS（特別是 continuous-flow LVAD）被作為心臟移植之外的替代療法。本來 LVAD 設計為過渡至移植的短期療法，但現在可在需長期等待移植或不符合移植條件病人使用數月至數年；因年紀太大或不符合移植而裝置永久性 LVAD 的病人數持續成長，終生 LVAD 治療已有可能性。

  • 沒有嚴重右心失能並嚴重三尖瓣脫垂

表 10-3　心臟移植適應症與禁忌症

可考慮心臟移植

  • 末期心臟衰竭合併嚴重症狀和不良預後，且無其他替代治療選項
  • 有動力、情緒穩定、病人應被提供完整資訊能配合術後的特別治療

禁忌症

  • 進行性感染
  • 嚴重周邊血管或腦血管疾病
  • 無法以藥物逆轉之肺動脈高壓（需考慮 LVAD，再重新評估是否可作為移植候選）
  • 癌症（與腫瘤科專家合作評估病人復發風險）
  • 無法回復之腎功能損傷
  • 影響多重器官之全身性疾病、其他預後不佳之嚴重共病
  • 移植前 BMI > 35 kg/m²（建議減重至 BMI < 35 kg/m²）
  • 酒精或藥物濫用
  • 社會支持系統不足，無法配合後續門診照護

初步評估

  1. 個人病史：高血壓、心血管疾病、使用心毒性藥物、放射線接觸史、端坐呼吸、陣發夜間呼吸困難、旅遊接觸史
  2. 理學檢查：血壓、心跳、心氧、盜汗、意識不清、心雜音、肺囉音、頸靜脈怒張、下肢水腫、心尖搏動位置改變、奇脈、肝腫大
  3. 心電圖檢查異常

血液檢驗項目

  1. NT-proBNP > 900 pg/mL
  2. 心肌酵素：CK、CK-MB、Troponin-T
  3. 血液常規計數檢驗（CBC）
  4. 生化檢驗：肝功能（AST、ALT、T-Bil）、腎功能（BUN、Cre）、甲狀腺功能（TSH、Free T4）、血脂（lipid profile）、尿酸、血糖（Glucose、HbA1c）等

裝置之評估

  1. 評估 CRT-P：NYHA Fc > II、左心室射血分率 ≤ 35%、左束支傳導阻滯且 QRSd > 150 毫秒（證據等級 I）
  2. 評估 ICD 或 CRT-D：經心衰竭治療、心肌梗塞治療 40 天後仍左心室射血分率 ≤ 35% 且 VT 或 VF（證據等級 I）
不穩定定義
  1. 血壓 < 90 mmHg
  2. 血氧 < 90%
  3. 心跳 < 50 或 > 120 次/分
  4. 心室顫動／心室頻脈
  5. 休克
  6. 意識不清
  7. 呼吸 > 30 次/分
